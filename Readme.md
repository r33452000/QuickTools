# July Tu的工具箱 (My QuickTools)

這是我個人開發的實用工具集，主要用於解決日常工作和學習中遇到的各種問題。

## 工具清單

### 1. 嘸蝦米加字加詞多行支援工具 (BoshiamyMult)

#### 功能
- 讓嘸蝦米輸入法的加字加詞功能支援多行文字
- 自動處理換行符號，方便輸入多行內容

#### 使用說明
1. 開啟 [index.html](./BoshiamyMult/index.html) 網頁
2. 在輸入框中輸入您想要設定的多行文字
3. 系統會自動處理換行符號，生成適合嘸蝦米加字加詞的格式
4. 複製結果並貼到嘸蝦米的加字加詞工具中

### 2. PromptBookmark - 書籤式 Prompt 管理器

#### 功能特色
- 將常用的 AI Prompt 轉換為瀏覽器書籤，點擊即可一鍵複製到剪貼簿
- 用瀏覽器書籤管理器整理 Prompt，支援分類、編輯、刪除
- 支援中英雙語介面，採用便條紙風格的深色主題設計
- 免安裝、免註冊，資料完全在本地處理，適用於所有現代瀏覽器

#### 使用說明
1. 開啟 [index.html](./PromptBookmark/index.html) 網頁
2. 輸入 Prompt 標題和內容
3. 點擊「產生書籤」按鈕
4. 將產生的書籤拖曳到書籤列或右鍵加入書籤
5. 使用時點擊書籤即可複製 Prompt 內容

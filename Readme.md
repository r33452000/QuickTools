# July Tu的工具箱 (My QuickTools)

這是我個人開發的實用工具集，主要用於解決日常工作和學習中遇到的各種問題。

## 工具清單

### 1. 嘸蝦米加字加詞多行支援工具 (BoshiamyMult)

#### 功能
- 讓嘸蝦米輸入法的加字加詞功能支援多行文字
- 自動處理換行符號，方便輸入多行內容

#### 使用說明
1. 開啟 [index.html](./BoshiamyMult/index.html) 網頁
2. 在輸入框中輸入您想要設定的多行文字
3. 系統會自動處理換行符號，生成適合嘸蝦米加字加詞的格式
4. 複製結果並貼到嘸蝦米的加字加詞工具中
